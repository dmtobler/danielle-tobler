Online Ocarina

Feb 2020

This project employed JavaScript with jQuery, event listeners, and general DOM manipulation to render a "playable" online ocarina based on The Legend of Zelda: Ocarina of Time (OoT) (Nintendo 1998). Users are able to produce notes through pressing the up, down, left, and right arrow keys as well as the "A" key. If an OoT song sequence is correctly played, users will receive an audio-visual confirmation followed by an audio clip of the song in question, emulating native gameplay.
